🧠 About the AI Model

CNN + Bidirectional LSTM hybrid trained on the Coswara dataset.

Architecture: 2D CNN → Reshape → BiLSTM × 2 → Dense → Softmax
Input Features: MFCC (40 coeff) + Delta + Delta² → 40×216×3 tensor
Training Data: Coswara, COUGHVID, ICBHI Respiratory Sound Database
Dataset Classes: Healthy, COVID-19, Asthma, Bronchitis, Pneumonia, URTI, COPD
